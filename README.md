# SpringBoot-Learning
Internal Working Of SpringBoot :
 Today I studied some important Spring Boot annotations like . @Component, @Autowired, @SpringBootApplication, @ConditionalOnProperty, and @Override. 
 I learned what each annotation does and how it is used in a Spring Boot application.
 1) SpringBootApplication => Marks the main class and enables Spring Boot features.
 2) @Component => Tells Spring to create and manage an object (bean) of the class.
 3) @Autowired => Automatically injects the required dependency.
 4) @ConditionalOnProperty  => Creates a bean only when a specific property has the required value.
